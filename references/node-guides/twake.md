# Twake — `n8n-nodes-base.twake`
**Type** `n8n-nodes-base.twake` · **typeVersion** 1 · **action**
**What:** Send messages to a Twake channel.
**Credentials:** `twakeCloudApi` (Twake Cloud) or `twakeServerApi` (self-hosted).
**Resources / Operations:**
| Resource | Operation |
|----------|-----------|
| Message | Send |

**Key params & gotchas:** Requires channel ID (not name). Self-hosted Twake needs a separate credential type with server URL.
**Source:** n8n-nodes-base.twake.md  [doc-verified]
