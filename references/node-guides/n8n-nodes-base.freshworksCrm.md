# Freshworks CRM — `n8n-nodes-base.freshworksCrm`
**Type** `n8n-nodes-base.freshworksCrm` · **action**
**What:** Manage Freshworks CRM (formerly Freshsales) — accounts, appointments, contacts, deals, notes, tasks, and sales activities.
**Credentials:** freshworksCrmApi (API key + domain).

## Resources / Operations
| Resource | Operations |
|---|---|
| Account | Create, Delete, Retrieve, Retrieve All, Update |
| Appointment | Create, Delete, Retrieve, Retrieve All, Update |
| Contact | Create, Delete, Retrieve, Retrieve All, Update |
| Deal | Create, Delete, Retrieve, Retrieve All, Update |
| Note | Create, Delete, Update |
| Sales Activity | Retrieve, Retrieve All |
| Task | Create, Delete, Retrieve, Retrieve All, Update |

## Key params & gotchas
- **Sales Activity** is read-only (no create/update) — activities are auto-generated by Freshworks CRM based on user actions.
- **Note** has no Get operation — notes are linked to contacts/deals but cannot be retrieved independently via this node.
- Contact and Account IDs are numeric; use Retrieve All with filters to look them up by email or name.
- "Operation not supported" error indicates a plan-restricted feature.
